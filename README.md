# Happy 2-Month Anniversary Website

A React-based interactive anniversary website with:
- Home, Our Story, Gallery, Love Notes, Interactive, Wishlist, Secret, and Final Message pages
- Animations, gallery, handwritten fonts, heart cursor, starry background, music toggle, and more

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Folder Structure
- `src/pages/` — Each main page as a React component
- `src/components/` — Shared UI components
- `src/utils/` — Utilities (e.g., Konami code)
- `src/assets/` — Images, audio, etc.
- `src/styles/` — Global and page styles

## Customization
- Replace placeholder images, fonts, and audio in `src/assets/`
- Add animations and interactive features as described in each page's TODO

## Features
- Parallax, animated backgrounds, heart cursor
- Timeline, gallery, lightbox, typewriter, quizzes, Easter eggs
- LocalStorage for interactive features
- Music toggle
